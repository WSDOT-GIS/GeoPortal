Location Information

  1. Select a buffer distance and measurement unit.
  2. Select the data layers that you want to query using the checkboxes.
  3. Click on one of the following buttons:
    • Point
    • Line
    • Polygon
  4. Move your cursor over the map and follow the on screen instructions to draw a geometry.
  5. When you have finished drawing the geometry, a query will be sent to a web server.
  6. When the query has completed, a geometry will appear on the map. You can see the results of this query by clicking on the geometry.

The results from this tool are stored in three graphics layers in the web map (one for each type of geometry). To clear these graphics, click the "Clear Results" button on the Location Information tool.
