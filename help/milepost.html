Find Milepost

This tool can be used to locate a specific state route and milepost combination.

  1. Enter a Washington State Route Identifier. (Route numbers are always three digits. E.g., State route 5 should be entered as "005".)
  2. Specify whether this location lies on the increasing or decreasing side of the route.
  3. Select the milepost type: ARM or SRMP.
  4. Enter a value into the Milepost box.
  5. If you have specified SRMP, check the Back checkbox if you are using back mileage, or uncheck it if you are not.
  6. Select a Reference Date.
  7. Click the Find Milepost button.

Find Nearest Milepost

This tool will find the nearest point on a state route to a mouse click.

  1. Enter a search radius into the Radius box.
  2. Click the Find button.
